# SPRINTS_INDEX.md — HANDYMAX · Multimax Despacho

Índice oficial del desarrollo por Sprints incrementales (metodología vigente desde el Sprint 3.1). Cada Sprint migra una sección específica de `Multimax_Despacho_v1.3.html`. Este archivo se actualiza al finalizar cada Sprint — ver `docs/sprints/sprint-X.Y.md` para el detalle de cada uno.

**Desde el Sprint 3.2**, cada Sprint se identifica por el bloque/selector real del HTML (no por un nombre genérico de sección como "Sidebar"/"Navigation"/"Dashboard"). La columna "Elemento" de los Sprints 3.15 en adelante todavía usa los nombres genéricos originales porque no se ha hecho el análisis previo obligatorio de esos bloques — se actualizarán al selector real del HTML cuando le corresponda su turno, sin adelantar ese análisis ahora. El Sprint 3.4 confirmó, por inspección directa del HTML (no por el nombre genérico que tenía esta tabla, "Main Layout"), que el bloque real pendiente era `mx-suc-sel`; el Sprint 3.5 verificó que "Publish Modal" sí correspondía al bloque real (`PublishModal`); el Sprint 3.6 determinó que "Job Cards" **no** correspondía al bloque real reconstruible ahora mismo; el Sprint 3.7 confirmó que "Radar" **sí** es un nombre real (`function Radar`), pero descartó la arquitectura multi-componente que sugería el brief; el Sprint 3.8 confirmó que "Countdown"/`CountRing` **sí** es un nombre real, y reportó que existe un segundo componente de countdown distinto (`LiveCountdown`) sin Sprint asignado en ese momento; el Sprint 3.9 asignó y reconstruyó ese componente (`LiveCountdown`), descartando el nombre genérico "Timeline" que traía esta tabla; el Sprint 3.10 determinó que "Installer Dashboard" **no** es el nombre de ninguna función real — el equivalente real es `Installer(props)`, del cual se reconstruyó únicamente el subconjunto alcanzable sin invadir los Sprints ya reservados 3.11 (`InstallerProfile`)/3.12 (`InstallerJobs`); el Sprint 3.11 confirmó que "Installer Profile" **sí** corresponde exactamente a una función real (`function InstallerProfile({ meInfo })`) — a diferencia de la mayoría de los Sprints anteriores, sin corrección de nombre necesaria; el Sprint 3.12 confirmó que "InstallerJobs" **sí** corresponde exactamente a una función real (`function InstallerJobs()`); el Sprint 3.13 determinó que "Admin Dashboard" **no** es el nombre de ninguna función real — el equivalente real es `AdminPanel()`; el Sprint 3.14 confirmó que "Calendar" **sí** corresponde exactamente a una función real (`function MasterCalendar()`) — ver notas abajo.
